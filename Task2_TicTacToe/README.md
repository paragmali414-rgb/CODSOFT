# Tic Tac Toe AI 🎮

This project is developed as **Task 2 for the CODSOFT Artificial Intelligence Internship**.

The application allows a human player to play Tic Tac Toe against an AI agent.  
The AI uses the **Minimax Algorithm**, which ensures optimal moves and makes the game challenging.

---

## 📌 Features

- Human vs AI gameplay
- AI powered by Minimax algorithm
- Interactive user interface
- Restart game functionality
- Moderate difficulty (AI sometimes plays randomly for fairness)

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Minimax Algorithm (Game AI)

---

## 🎯 How It Works

The AI evaluates all possible moves using the **Minimax algorithm**, assigning scores to different board states and selecting the best move.

- AI Win → Score +10  
- Human Win → Score -10  
- Draw → Score 0  

This ensures the AI makes the most optimal decision.
