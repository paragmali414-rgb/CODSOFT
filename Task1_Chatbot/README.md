# Task 1 - Rule-Based Chatbot
CodSoft Artificial Intelligence Internship

## Objective
The objective of this task is to build a simple rule-based chatbot that responds to user inputs using predefined rules and conditional statements.

## Project Description
This chatbot interacts with users and provides responses based on keyword detection and conditional logic.

The chatbot can:
- Respond to greetings
- Provide information about Artificial Intelligence
- Provide internship-related information
- Show current date and time
- Perform basic calculations
- Detect simple user sentiments (happy/sad)
- Store and recall user's name

This project demonstrates basic Natural Language Processing concepts and conversation flow management using rule-based logic.

## Technologies Used
- Python
- Random Module
- Datetime Module

## How to Run the Project
1. Install Python on your system.
2. Download the repository.
3. Navigate to Task1_Chatbot folder.
4. Run the following command:

   python chatbot.py

## Learning Outcomes
- Understanding rule-based AI systems
- Using conditional statements effectively
- Handling user input in Python
- Structuring a simple conversational flow
