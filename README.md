# 🤖 CODSOFT AI Internship Projects

This repository contains all tasks completed as part of the CODSOFT Artificial Intelligence Internship.

---

## 📌 Tasks Completed

### 🔹 Task 1: Chatbot with Rule-Based Responses
A simple chatbot that responds using predefined rules and conditional logic.

### 🔹 Task 2: Tic-Tac-Toe AI
An AI-based Tic-Tac-Toe game using the Minimax algorithm to play optimally.

### 🔹 Task 3: Image Captioning
An AI application that generates captions for images using a pre-trained BLIP model (combining Computer Vision and NLP).
