# 🖼️ Image Captioning using BLIP

This project is part of CODSOFT AI Internship Task 3.

It generates captions for images using a pre-trained BLIP model.

## 🚀 Features
- Upload image
- Generates:
  - Short Caption
  - Detailed Caption
- Built with Streamlit

## 🛠️ Tech Stack
- Python
- Streamlit
- Transformers (HuggingFace)
- Torch

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app.py
